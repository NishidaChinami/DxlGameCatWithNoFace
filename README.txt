【作品について】
・本作品のコードで使用しているLibraryの中のtnl-から始まるコードは、学校の講師によって配布されたものであり、そちらを一部使用して作品制作を行いました。
・今回の作品では、PlayerやEnemyなど各GameObjectそれぞれにManagerを持たせ、インスタンス化を行っておりますが、Factoryパターンなどの実装で見やすくすることを考えています。
・本作品のコーディングでは、クラス同士の依存度が高くなっていますが、次の作品ではMediatorパターンの実装を試みたいと考えています。

【作品のこだわり】
・作品で一番こだわったポイントは、ギミック等の実装です。横スクロールアクションでよくみられるロープは、プレイヤーが振り子のような挙動になるよう意識し、また縄から離れたときに縄の動きに減衰がかかるようにしました。また、プレイヤーの横移動には慣性をつけました。さらに、プレイヤーが箱を動かしてその上に乗れるようなギミックの仕掛けを作りたかったため、側面に触れているときは動かせるような補正をしました。
・また、意識した点はオブジェクト指向に則ったクラス設計です。まだまだ、改善点は多いですが、シングルトンパターンやストラテジーパターンなどパターン設計の導入や親クラスと子クラスの関係を意識しながら設計しました。
